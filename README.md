# Algorand Project: Decentralized Voting System

## Objective
This repository contains my Algorand-based project for the course.  
The goal is to build a decentralized voting application using Algorand smart contracts.

## Tech Stack
- Algorand SDK (Python)
- PyTeal for smart contracts
- Algorand Sandbox for local testing
